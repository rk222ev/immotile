Sometimes some static files is all you need

Not all projects has the need for a server or the demands of a single page application. Sometimes all you need is a bunch of static files. Of course you don't want to have to handle html or javascript. You want to use all the niceness available in the Clojure(script) community. Thats where Immotile comes in.

Heavily inspired by Static.

Why and where should I use Immotile?

Immotile's mission is to be a static site generator using the tools that more or less has become a standard in Clojure development. While at the same time trying to keep up with new and simpler idea's that makes development of static sites easier and hopefully a bit more enjoyable. Filling the gap in the Clojure(script) world between fully blown server rendered and single page applications.

The core features are a fast feedback loop and automation.
